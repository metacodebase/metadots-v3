# Required Images

The following images need to be added to this directory:

## Header/Footer Logos
- `metadots-logo.svg` - Main Metadots logo (used in header and footer)
- `clutch.png` - Clutch.co logo (used in footer social links)

## Client Logos (used on homepage)
- `BoiceVox-logo.png`
- `spectrum-logo.png`
- `haulos-logo.svg`
- `fangenie-logo.png`
- `teammatch-logo.png`

## Notes
- All images should be placed directly in this `/public/images/` directory
- In Next.js, files in the `public` folder are served from the root path
- So `/images/metadots-logo.svg` maps to `public/images/metadots-logo.svg`
